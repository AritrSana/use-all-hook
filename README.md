# "use-all-hook" is a react hooks package for a lot of hooks in the future.

## Installing

###### If using npm

```
npm i use-all-hook
```

###### If using yarn

```
yarn add use-all-hook
```

## Require them

```javascript
import AllHooks from "use-all-hook";
```

## or

```javascript
import { useAsync, useFetch } from "use-all-hook";
```
